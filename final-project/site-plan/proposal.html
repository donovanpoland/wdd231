Proposal

Recipe Explorer

Subject

Recipe Explorer - a dynamic, responsive website that allows users to discover, view, and save recipes with details such as ingredients, preparation time, and instructions.

Description

Recipe Explorer will be a three-page website that introduces users to a variety of recipes.

  • The Home Page will provide a welcoming introduction and highlight featured recipes.
  • The Recipes Page will dynamically fetch and display recipes from an API called The meal DB. Each recipe will show details like title, ingredients, prep time, and an image.
  • The Favorites Page will allow users to save and view their favorite recipes using local storage.
  • On the Favorites page form will let users submit their own recipe ideas, with the data displayed on a submission acceptance page.
  • A modal dialog will be included where recipes are displayed so users can view full recipe details without leaving the current page.

Purpose and Reason

I chose this project for several reasons. First, I wanted to challenge myself by working with APIs, and food is a naturally dynamic dataset because of its variety and flexibility. Also, as I've been growing in my web development skills, my mom has asked me to create a website for her to share her recipes and offer food courses. This project will serve as both practice and a proof of concept, allowing me to experiment with features that could eventually be used in her real site.
